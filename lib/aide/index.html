Valhalla::Catalog
Aide à l'utilisation

1. Navigation
La zone de navigation apparaît dans les sous-dossiers. Elle permet de retourner au niveau supérieur (c'est-à-dire au dossier parent du dossier actuellement affiché), de consulter l'arborescence complète de fichiers et des dossiers, et de connaître rapidement le nom du dossier dont le contenu est actuellement affiché.

2. Sous-dossiers
La zone d'affichage des sous-dossiers affiche tous les dossiers enfants contenus dans le dossier actuellement affiché. Le nombre entre parenthèses correspond au nombre d'éléments (fichiers et dossiers) contenus dans le sous-dossier. Cliquez sur le nom d'un sous-dossier pour en afficher le contenu.

3. Fichiers
La zone d'affichage des fichiers présente la liste des fichiers contenus dans le dossier actuellement affiché. La première colonne contient une image (terre et flèche verte) : cliquez sur cette image pour télécharger le fichier. Les autres colonnes présentent les informations que l'administrateur du catalogue a choisi d'afficher. Lorsqu'une information n'est pas disponible pour un fichier, la case correspondante est remplie avec un point d'interrogation (?). Sont indiqués en dessous de la liste des fichiers : le nombre de fichiers présents dans le dossier et le poids total de ces fichiers.

4. Ajout de fichiers
Le formulaire d'ajout de fichiers permet de mettre des fichiers en ligne. Les champs définis par l'administrateur du catalogue sont à remplir, ils représentent les méta-données associées à chaque fichier. Si un champ n'est pas rempli, la case correspondante dans la liste des fichiers sera laissée vide. Il est possible de mettre en ligne plusieurs fichiers en même temps ; tous ces fichiers auront alors les mêmes méta-données. Pour envoyer plusieurs fichiers en même temps, cliquez autant de fois que nécessaire sur le bouton "+" et choisissez les fichiers désirés. Une fois le formulaire rempli, cliquez sur le bouton d'envoi et patientez pendant que les fichiers sont transmis (cela peut être long : le temps d'envoi dépend de la vitesse de votre connexion et de la taille des fichiers envoyés).

5. Nouveau dossier
Le formulaire nouveau dossier permet de créer des sous-dossiers dans le dossier actuellement affiché.
